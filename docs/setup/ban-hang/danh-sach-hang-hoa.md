# **Thêm hàng hóa**

Dưới đây là những hướng dẫn thao tác cơ bản trên phần mềm bán hàng M-invoice vô cùng mạch lạc và dễ hiểu.

## **Hướng dẫn thêm hàng hóa**

???+ Note "Ghi chú"

    Thêm hàng hóa vào phần mềm để quản lý, xuất hóa đơn nhanh chóng.
